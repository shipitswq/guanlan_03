# 2026-07-23

## 量化交易平台 V1 — 全栈搭建

### 架构
- **后端**: Python FastAPI + WebSocket + SQLite
- **前端**: React 18 + TypeScript + Vite + Ant Design + Recharts
- **数据源**: 通达信(mootdx TCP) + 腾讯财经估值数据

### 功能模块
1. **策略设计** (页面: /strategies): 内置3种策略(双均线/RSI/MACD)，支持创建自定义策略
2. **回测引擎** (页面: /backtest): 支持A股标的回测，计算收益率/夏普/最大回撤/胜率/盈亏比，净值曲线可视化
3. **模拟交易** (页面: /trading): 注册策略到标的，定时轮询行情，自动执行开平仓，持仓管理
4. **实时Feed** (页面: /feed): WebSocket推送交易事件(signal/order/position/pnl)，支持客户端订阅

### 文件结构
```
backend/
├── main.py          # FastAPI主入口，CORS/WebSocket/SSE
├── data/service.py  # 数据服务层(mootdx/腾讯/东财)

├── engine/          # 回测引擎 + 模拟交易引擎 + Feed管理器

frontend/
└── src/

    ├── components/  # AppLayout
    ├── pages/       # Dashboard/StrategyDesigner/Backtest/Trading/Feed

```
